title:      Eteindre l'ordinateur avec un timer
lang: fr
categories: linux
---

Pour éteindre l'ordinateur à une heure précise
```bash 
$ sudo shutdown -P 23:30
``` 

Pour éteindre l'ordinateur dans 30 minutes
```bash 
$ sudo shutdown -P 30
``` 

Dans les 2 cas, ne pas fermer le terminal.

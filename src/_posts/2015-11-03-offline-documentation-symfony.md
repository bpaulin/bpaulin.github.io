Téléchargement de la doc pour symfony 2.7, pour une utilisation hors ligne

```bash
#!/bin/bash
wget http://symfony.com/pdf/Symfony_book_2.7.pdf -O Symfony_book_2.7.pdf
wget http://symfony.com/pdf/Symfony_cookbook_2.7.pdf -O Symfony_cookbook_2.7.pdf
wget http://symfony.com/pdf/Symfony_components_2.7.pdf -O Symfony_components_2.7.pdf
```
